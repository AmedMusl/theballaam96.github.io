Practice ROM 1.3.3 Features List

Since 1.4 is going to involve a large scale controls/UI overhaul, I haven't really put much time into a features list and something official for 1.3.3 isn't reallly worth it all things considered.
That being said, I understand the desire for a features list/controls for 1.3.3 until 1.4 comes out, so here is a basic HTML page for that

Interface

  • Open up the pause menu and press D-Up to access a tools menu that allows you to modify various in-game aspects
    • To close, press D-Up again or close the pause menu with Start/B
  • Use the analog stick to navigate the menus
  • Use the A Button to active the functionality of a menu item (eg. A to warp to map, A to toggle flag state, A to set watch)
  • Use the L Button to speed up scrolling speed
  • To change "screens" (essentially just a way to segment controls), press Z/R
    • The screens are as follows:
      • Screen 1: Set 'Important Flags', Savestates, Watches/Miscellaneous items
      • Screen 2: Warp to map
      • Screen 3: Control how the timer words
      • Screen 4: Toggle flags for individual bananaports
      • Screen 5: File Status, Cheats

Savestates

  • Savestates aren't 100% the savestates you may be used to with Emulators/Wii U Virtual console. They store some key variables that emulate most of what users want savestates for
  • To save a state, select "Save State"
  • To load a state, select "Load State" (This cannot be performed until a state has been saved)
    • There are two variants of this
      • "From Position" loads you back in from the map and position where you saved your savestate
      • "From Exit" loads you back in from the entrance you entered the map from where you saved your savestate
  • Savestate functionality is prevented in certain maps
    • See "Warping Preventions"
  • The following variables are stored with Savestates:
    • Kong played
    • Permanent flags (Golden Bananas, Crowns, Fairies, Boss Keys etc.)
    • Temporary flags (Fungi Bean, Long Boss Intros watched etc.)
    • Kong moves/guns/instruments
    • Kong coin count
    • Coloured Bananas and Golden Bananas for each kong for each level
    • Inventory items (Ammo, Crystals, Film etc.)

Warp to Map

  • Allows you to warp to most maps in the game
    • Maps that aren't in are either useless to speedrunners (DK Rap, Test Map, Nintendo Logo), or cause crashes due to technical reasons (See "Warping Preventions")
  • Each map is segmented into a level (eg. Dungeon will be in "Castle", Llama Temple will be in "Aztec")
  • In some cases, you can warp to a specific entrance for maps that are quite large, these will be stated in the map name

Timer Control

  • The timer can be controlled automatically by game events
  • Start Events:
    • On L Press
    • On input
    • On Map Load
  • End Events:
    • On L Press
    • On Transition (Map Transition into a new map)
    • On Cutscene being active
    • On GB Dance (Doesn't activate for dance skips)
  • Pause on Pause:
    • Toggles whether the pause the timer when the pause menu is pulled up
  • L will always act as master control of the timer

Displays (Watches)

  • Display a readout of an important game variable
  • Current watches available
    • Lag
    • Cumulative Lag
      • Press L to reset the counter
    • Horizontal Speed
    • A Timer
    • The timer which controls Giant Kosha's (Crystal Caves) actions
    • Movement State
    • Angle
  • If in K. Rool, the "K Rool Timer" will shift to the right to prevent the display being unreadable

Important Flags

  • Set/Clear/View the state of various important game flags that are of note. These aid in allowing the user to generate a specific scenario for practice

Position Warping

  • Allows you to store and warp to a specific position of your choosing
  • Press D-Right to store a position
  • Press D-Left to warp to a previously stored position
    • You can only have one position stored, and you can only warp to that position if you are in the correct map
  • Position Warping stores/loads the following variables:
    • X, Y and Z Co-ordinates
    • 'Stored Positions' (In the context of TBS. If you don't know what this is, don't worry about it)
    • Facing angle
    • Z Rotation (AKA "Skew Angle")
    • Velocity and Acceleration components in both the horizontal and vertical axis
    • Floor height

Tag Anywhere

  • Press D-Down to instantly change kongs to the next kong (DK > Diddy > Lanky > Tiny > Chunky)
    • NOTE: Tag Anywhere currently crashes if you perform it whilst you are either Enguarde or Rambi
  • If you have played Isotarge's dedicated Tag Anywhere ROM Hack, this is pretty similar to this, but with a lot of the "casual player protections" removed

Inventory Manipulation

  • Unlock Moves
    • Gives all of the moves in the game (Cranky/Candy/Funky/BFI Camera/Training Barrels) EXCEPT SIMIAN SLAM AND SNIPER SCOPE
      • See below on how to change Simian Slam/Sniper Scope
  • Slam Level
    • Changes the simian slam level between the four values
      • 0 = No Simian Slam
      • 1 = Regular Simian Slam (Green)
      • 2 = Super Simian Slam (Blue)
      • 3 = Super Duper Simian Slam (Red)
  • Sniper Scope
    • Toggles Sniper Scope on/off
      • Since sniper scope increases lag, this is useful for those wanting to practice runs where Sniper Scope isn't obtained
  • Restock Inventory
    • Restocks all of your inventory items (Ammo, Crystals, Film, Health)
  • Give Coins
    • Gives each kong 50 coins, more than enough for the player to purchase a specific moveset

File Status

  • 101% - Sets file status to be a 101% file in terms of collectable counts
  • Max% - Sets file status to be a Max% (132% / -124%) file in terms of collectable counts

Cheats

  • Infinite Health
    • Sets the health of the player to be full every frame. Doesn't prevent death in circumstances where the damage dealt exceeds the amount of health the player has
  • Auto-Phase State
    • Grants the player the "Phase State" (In the context of Phasewalking/falling/swimming)
    • In order to go through a wall, you will need to execute an action that would allow you to go through a wall under normal gameplay circumstances (eg. crouching)
  • L Mode
    • Various cheats (due to the nature of the cheat), can only be accessed by pressing L
    • L can only activate one cheat at a time
    • L Mode Cheats:
      • Stop Timer
        • AKA No cheats activated, L will always control timer
      • L to Levitate
        • Does what it says on the tin
        • Holding R doubles levitation speed
        • Holding Z halves levitation speed
      • Toggle Sandstorm
        • Toggles whether the sandstorm in Aztec is on or off (Only togglable in Aztec)
      • Cancel Cutscene
        • Immediately ends the cutscene that is currently playing

Toggle Individual Bananaports

  • Allows you to toggle whether a bananaport is tagged or not for each bananaport in the game

Setup Takeoff Skip

  • Sets the file up with the appropriate flags/variables set/cleared for practising "Takeoff Skip"

Warping preventions

  • For technical reasons, warping to and from certain maps is prohibited to prevent unintended behaviour/crashes
  • These maps are
    • The 7 Level Boss Fight Maps (Both Army Dillos, Both Dogadons, Mad Jack, Pufftoss, Kut Out)
    • Bonus Minigames (Bonus Barrel Minigames, Training Barrels, Helm Barrel Minigames)
    • Battle Crowns
    • Character Shops (Snide's, Funky's, Candy's, Cranky's)
    • Troff n Scoff
  • If you attempt to warp to/from one of these maps, you will here an error sound

Change Menu Variables

  • Aspect Ratio (4:3/16:9)
  • Music Level
  • SFX Level
  • Camera Mode (Free Cam/Follow Cam)

Various other quality of life features

  • Story skip is locked to on
  • Entire mystery menu is unlocked from the start
  • Game automatically skips the DK Rap and DKTV
  • Recoloured kongs
  • Options to disable position warping/Tag Anywhere

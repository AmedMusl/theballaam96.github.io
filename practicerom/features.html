Story Skip
Story Skip is set to be permanently on by default, which gets rid of any chance of forgetting to turn it on. Attempting to change this will set story skip back to on.
Mystery Menu
All permanent cartridge unlocks (that are awarded to the player during gameplay) are unlocked by default. This includes:
  • Cutscene Menu
    • Intro Story
    • All Level "K. Rool Intros"
    • K. Rool Press Button
    • K. Rool Takeoff
    • Game Over
    • End Sequence
  • Minigames Menu
    • Rambi & Enguarde Arena
    • DK Arcade
    • Jetpac
  • Bosses Menu
    • All 7 Level Bosses
    • The Final Battle (Unused beta feature)
  • Krusha Menu
  • Default Cheats Menu
  • Multiplayer Menu
Simple Tag Anywhere
Kongs can be changed with the press of the D-Down button unless the pause menu is open. This functionality can be manually disabled with the "Disable Tag Anywhere" toggle in the Tools Menu
Differences between the Practice ROM 'Tag Anywhere' and the dedicated 'Tag Anywhere' Mod
The goal of the 'Tag Anywhere' in the Practice ROM and the goal of the dedicated 'Tag Anywhere' ROM Hack are different. The Practice ROM's implementation is with the goal to support any glitches that can happen when a tag interrupts a game event. The dedicated 'Tag Anywhere' ROM Hack has more protections on when you can perform a tag to prevent disruptions to intended gameplay mechanics (usually these disruptions occur when a tag coincides with a game event).
Tools Menu Controls
                             
  Opens and closes the tools menu when the pause menu is pulled up
  Goes to previous tools menu screen
  Goes to next tools menu screen
  Select tools menu option
  Navigate up/down/left/right on the tools menu
  Changes left/right scrolling cooldown from 6 frames to 2 frames
  /   Exit out of the pause menu
Position Storage Controls
  Stores position information
  Loads position information if on the same map as the stored information
Information stored with position storage
Various pieces of information are stored when a position is stored with the Practice ROM. These pieces of information are loaded back in as soon as you load back up your saved position.
These variables are stored:
  • X, Y and Z Co-ordinates of the Kong
  • "Stored Positions"
    • In context of Tag Barrel Storage and Telegrabbing
  • Facing Angle
  • Horizontal & vertical speed components
  • Vertical acceleration (AKA gravity)
  • Floor height
  • Map
    • This is for the purpose of detecting whether you are in the same map when you attempt to load the position up
Map Warping Controls
The following controls only apply when you are on the map warp screen:
  (When highlighting level) Changes selected level
  (When highlighting map) Changes selected map
  Warp to selected map
